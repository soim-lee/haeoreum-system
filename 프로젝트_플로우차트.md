# 해오름인포텍 업무시스템 플로우차트

## 1. 시스템 전체 플로우

```mermaid
graph TD
    A[사용자 접속] --> B{로그인 여부}
    B -->|미로그인| C[로그인 페이지]
    C --> D[접근키 입력]
    D --> E{접근키 검증}
    E -->|실패| C
    E -->|성공| F[대시보드]
    B -->|로그인됨| F
    
    F --> G[메인 메뉴 선택]
    G --> H[프로젝트 관리]
    G --> I[이력서 관리]
    G --> J[지원자 관리]
    G --> K[계약 관리]
    G --> L[캘린더]
    G --> M[AI 에이전트]
    G --> N[블랙리스트]
    G --> O[양식 관리]
```

## 2. 프로젝트 관리 플로우

```mermaid
graph TD
    A[프로젝트 관리 페이지] --> B[프로젝트 목록 조회]
    B --> C{작업 선택}
    
    C --> D[새 프로젝트 등록]
    D --> E[프로젝트 정보 입력]
    E --> F[프로젝트명, 거래처, 기간, 금액]
    F --> G[저장]
    G --> H[프로젝트 목록 업데이트]
    
    C --> I[기존 프로젝트 수정]
    I --> J[프로젝트 선택]
    J --> K[정보 수정]
    K --> G
    
    C --> L[프로젝트 삭제]
    L --> M[삭제 확인]
    M --> N[프로젝트 삭제 실행]
    N --> H
    
    C --> O[프로젝트 검색/필터링]
    O --> P[검색 조건 입력]
    P --> Q[결과 표시]
```

## 3. 이력서 관리 플로우

```mermaid
graph TD
    A[이력서 관리 페이지] --> B[이력서 업로드]
    B --> C[파일 선택]
    C --> D{파일 형식 검증}
    D -->|실패| E[오류 메시지]
    D -->|성공| F[파일 업로드]
    
    F --> G[텍스트 추출]
    G --> H{추출 성공}
    H -->|실패| I[수동 입력 요청]
    H -->|성공| J[AI 분석 시작]
    
    J --> K[이름/연락처 추출]
    K --> L[학력/경력 분석]
    L --> M[기술 스택 인식]
    M --> N[산업 분야 분류]
    N --> O[등급 산정]
    O --> P[블랙리스트 확인]
    
    P --> Q{블랙리스트 해당}
    Q -->|예| R[차단 알림]
    Q -->|아니오| S[프로젝트 매칭]
    
    S --> T[매칭 점수 계산]
    T --> U[매칭 결과 저장]
    U --> V[이력서 카드 생성]
    V --> W[목록에 추가]
```

## 4. 지원자 관리 플로우

```mermaid
graph TD
    A[지원자 관리 페이지] --> B[지원자 목록 조회]
    B --> C[상태별 필터링]
    C --> D[지원/검토중/거절/합격]
    
    D --> E[지원자 선택]
    E --> F[상세 정보 확인]
    F --> G{상태 변경}
    
    G --> H[검토중으로 변경]
    H --> I[검토자 배정]
    I --> J[평가 진행]
    
    G --> K[거절 처리]
    K --> L[거절 사유 입력]
    L --> M[거절 완료]
    
    G --> N[합격 처리]
    N --> O[합격 확인]
    O --> P[자동 계약 생성]
    P --> Q[계약 정보 설정]
    Q --> R[계약 등록 완료]
    R --> S[캘린더 일정 추가]
    
    J --> T[평가 완료]
    T --> G
```

## 6. 캘린더 플로우

```mermaid
graph TD
    A[캘린더 페이지] --> B[현재 월 일정 조회]
    B --> C[일정 목록 표시]
    C --> D[계약 마감일 표시]
    D --> E[월간 캘린더 표시]
    
    E --> F{사용자 액션}
    
    F --> G[새 일정 추가]
    G --> H[일정 정보 입력]
    H --> I[제목, 날짜, 시간, 설명]
    I --> J[일정 저장]
    J --> K[캘린더 업데이트]
    
    F --> L[날짜 클릭]
    L --> M[해당 날짜 상세 모달]
    M --> N[일정 목록 표시]
    N --> O[계약 마감일 표시]
    
    F --> P[월 네비게이션]
    P --> Q[이전/다음 달 이동]
    Q --> R[해당 월 데이터 조회]
    R --> B
    
    F --> S[일정 수정/삭제]
    S --> T[일정 선택]
    T --> U{작업 선택}
    U --> V[수정]
    U --> W[삭제]
    V --> H
    W --> X[삭제 확인]
    X --> K
```

## 7. AI 에이전트 플로우

```mermaid
graph TD
    A[AI 에이전트 페이지] --> B[채팅 인터페이스 로드]

    D --> E{사용자 입력}
    
    E --> F[텍스트 질문 입력]
    F --> G[질문 전송]
    
    E --> H[빠른 질문 버튼 클릭]
    H --> G
    
    G --> I[질문 유형 분석]
    I --> J{분석 결과}
    
    J --> K[수익 관련 질문]
    K --> L[계약 데이터 조회]
    L --> M[매출 계산]
    
    J --> N[프로젝트 관련 질문]
    N --> O[프로젝트 데이터 조회]
    O --> P[프로젝트 정보 정리]
    
    J --> Q[이력서 관련 질문]
    Q --> R[이력서 검색]
    R --> S[검색 결과 정리]
    
    J --> T[계약 관련 질문]
    T --> U[계약 데이터 조회]
    U --> V[계약 정보 정리]
    
    M --> W[OpenAI API 호출]
    P --> W
    S --> W
    V --> W
    
    W --> X[AI 응답 생성]
    X --> Y[응답 표시]
    Y --> Z[추가 질문 대기]
    Z --> E
```

## 10. 배포 및 운영 플로우

```mermaid
graph TD
    A[개발 완료] --> B[코드 커밋]
    B --> C[Git 저장소 푸시]
    C --> D{배포 환경 선택}
    
    D --> E[Replit 배포]
    E --> F[Deploy 버튼 클릭]
    F --> G[자동 빌드]
    G --> H[서비스 시작]
    
    D --> I[Tomcat 배포]
    I --> J[서버 준비]
    J --> K[Java/Tomcat 설치]
    K --> L[PostgreSQL 설치]
    L --> M[프로젝트 빌드]
    M --> N[웹앱 배포]
    N --> O[백엔드 서버 시작]
    
    D --> P[Docker 배포]
    P --> Q[Dockerfile 빌드]
    Q --> R[컨테이너 실행]
    
    H --> S[서비스 모니터링]
    O --> S
    R --> S
    
    S --> T[로그 확인]
    T --> U[성능 모니터링]
    U --> V[오류 대응]
    V --> W[시스템 업데이트]
    W --> A
```

이 플로우차트들은 해오름인포텍 업무시스템의 전체적인 작업 흐름과 각 기능별 상세 프로세스를 시각적으로 보여줍니다. 개발팀의 이해와 유지보수에 도움이 될 것입니다.
